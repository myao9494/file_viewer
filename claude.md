# ファイルビューア 開発仕様書 (claude.md)

## プロジェクト概要
ローカルファイルをブラウザで閲覧・操作するための Flask ベースのアプリケーション。

## 開発ルール
- **言語**: 日本語を基本とする。
- **実装方針**: テスト駆動開発 (TDD) を原則とする。
- **ドキュメント**: `docs/` フォルダ内に詳細ドキュメントを作成し、`claude.md` には全体像を簡潔に記述する。
- **コメント**: 各ファイルの冒頭に日本語で仕様を記述する。

## 主要機能
- ファイル/ディレクトリのブラウズ
- 各種ファイル（Markdown, CSV, 画像, PDF 等）のプレビュー
- 外部エディタ（VS Code / Cursor）でのオープン
- 特定の拡張子（.excalidraw）を専用エディタ（Port 3001）で開く機能

## 進行中の変更
- `.excalidraw.md` 拡張子のファイルを Port 3001 の Excalidraw エディタで開くように拡張。
